
# 🚗 CarPlate Master MVP

차량 번호판 관리 및 조회를 위한 하이브리드 웹/앱 서비스 MVP 모델입니다.

## ✨ 주요 기능
- **차량 조회 시스템**: 번호판 뒷자리(4자리) 입력을 통한 빠른 차주 정보 매칭.
- **관리자 패널**: 차량 데이터 CRUD 및 CSV 대량 데이터 업로드.
- **보안**: 관리자 승인 기반의 회원가입 시스템 시뮬레이션.

## 🚀 Vercel 배포 가이드 (중요)
Vercel에서 배포할 때 아래 설정을 확인하세요:
1. **Framework Preset**: `Vite` 선택 (자동으로 선택됨)
2. **Build Command**: `npm run build`
3. **Output Directory**: `dist`
4. **Install Command**: `npm install`

## 💻 로컬 개발 환경 실행
```bash
npm install
npm run dev
```
